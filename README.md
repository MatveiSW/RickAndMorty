# Возможности 
- Просмотр персонажей RickAndMorty
- Листать список персонажей
- Просмотр информации о персонаже
- Узнать в каких эпизодах снимался персонаж

# Я использовал: 
- UIKit
- Auto layout
- URLSession, Kingfisher
- GCD
- MVC
